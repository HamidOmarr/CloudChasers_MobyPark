# MobyPark Parking API - CloudChasers

## Overview
The MobyPark Parking API provides a comprehensive solution for managing parking services, including functionalities for parking space management, reservations, payments, and user authentication.

## Setup
To set up the MobyPark Parking API, follow these steps:
1. Clone the repository
2. Ensure Docker is installed on your host machine.
3. Navigate to the project directory.
4. Set up environment variables as needed.
5. Run the Docker container using the provided `Dockerfile`.
6. Access the API using the configured host and port.

## API Documentation
The API documentation is available in OpenAPI (Swagger) format. You can access the Swagger UI at `{server_url}/swagger/index.html` once the server is running.
The main specification file can be accessed at `{server_url}/swagger/v1/swagger.json`.
For generating separated and organized Swagger documentation, refer to the swagger.md file in the OpenAPI directory.
